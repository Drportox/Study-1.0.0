Ações e Interações com Tailwind CSS

Vamos explorar as diferentes interações que podem ser aplicadas a elementos usando Tailwind CSS. Aqui estão os exemplos:

Exemplo de HOVER: Quando o mouse passa sobre o botão, a cor de fundo muda dependendo do tamanho da tela. Para telas médias ou maiores (md), o botão terá a cor azul claro ao passar o mouse.

Exemplo de ACTIVE: Quando o botão está sendo clicado, a cor de fundo muda para verde claro.

Exemplo de FOCUS: Quando o campo de texto recebe o foco (ao clicar ou navegar até ele), a cor de fundo muda para verde claro.

Exemplo de DISABLED: Quando o campo de texto está desabilitado, a cor de fundo muda para vermelho escuro.

Exemplo de Estilo para Upload: Personalize o botão de upload com uma cor de fundo diferente para o estado inicial.
